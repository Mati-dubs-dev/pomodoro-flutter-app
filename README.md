# Pomodoro Flutter App

A clean and simple **Pomodoro timer application** built with Flutter to help users stay focused and manage their time effectively.

This app follows the Pomodoro Technique, a popular productivity method that alternates focused work sessions with short breaks.

---

## Features

* 🍅 Pomodoro focus timer
* ⏸ Short and long breaks
* 📊 Session tracking and statistics
* 🔔 Notifications when sessions end
* 📱 Clean and minimal user interface
* 💾 Local data persistence
* 📈 Weekly statistics chart

---

## Screens

The application includes the following main screens:

* **Pomodoro Screen** – Main focus timer
* **Statistics Screen** – Weekly productivity chart
* **Settings Screen** – Customize session durations

---

## Technologies Used

* **Flutter**
* **Dart**
* **Riverpod** (State Management)
* **Local Storage**
* **Custom UI Widgets**

---

## Project Structure

lib/
models/ → Data models
providers/ → State management with Riverpod
services/ → App services (timer, storage, notifications)
screens/ → Application screens
widgets/ → Reusable UI components
utils/ → Utility functions

---

## Getting Started

To run this project locally:

1. Clone the repository

```
git clone https://github.com/Mati-dubs-dev/pomodoro-flutter-app.git
```

2. Navigate to the project folder

```
cd pomodoro-flutter-app
```

3. Install dependencies

```
flutter pub get
```

4. Run the app

```
flutter run
```
